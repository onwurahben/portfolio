DocQuery Assistant

Role

AI Systems Engineer

Focus

RAG & Information Retrieval

Stack

Python, OpenAI, Pinecone

The Challenge

Internal documentation at large organizations is often fragmented across multiple platforms, making it nearly impossible for employees to find specific answers quickly.

The Solution

I developed DocQuery, a retrieval-augmented generation (RAG) system that indexes internal docs into a vector database. Using advanced reranking and semantic search, it provides instant, cited answers to complex queries.

Key Metric

70% reduction in search time

Infrastructure

Deployed as a scalable microservice with FastAPI and Docker.
